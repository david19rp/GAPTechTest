Steps to install the repo:

 1. clone the repo.
 2. once is install in the terminal type: npm i
 3. Create the folders as per the SS below:
    https://raw.githubusercontent.com/david19rp/gulp_Front-end_Boilerplate/master/structure.png

    Or you can follow the gulpfile.js and create the folders and files.

    (in Mac terminal type : "touch" to create files and mkdir to create folders)

- Gulp
